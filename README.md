## AceTipper

## Setup

1. [Install `ace-tipper`](https://www.npmjs.com/package/ace-tipper) globally
    * `npm install ace-tipper --global`
2. `cd` in to the newly created app
    * `cd ace-tipper`
3. Install dependencies
    * `npm install`
4. Start the app
    * `npm start`
5. Open a browser to `http://localhost:3000/`
6. Done
